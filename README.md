## AudioPlay

Anime Theme Audio Player

## Tech Stack

**Client:** React

**Server:** Tauri, Sqlite

<img src="https://skillicons.dev/icons?i=tauri,react,ts,rust,sqlite" alt="stack">

## Run Locally

Clone the project

```bash
  git clone https://github.com/Corro-Kun/Audioplayer.git
```

Go to the project directory

```bash
  cd Audioplayer
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run tauri dev
```
